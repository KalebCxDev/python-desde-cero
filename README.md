# Curso de Python Desde Cero
Este es un proyecto donde se aprenden los fundamentos de Python. El proyecto está dividido en varias secciones o temas:

## Estructura del proyecto
```
python-desde-cero
│
└───tema                            # Nombre del tema
    │   README.md                   # Explicacion de los apuntes y ejercicios vistos en el tema
    │
    ├───apuntes                     # Archivos de apuntes
    │       00_archivo_tema1.py
    │       01_archivo_tema.py
    │
    ├───ejercicios                  # Ejercicios sobre el tema
    │       01_ejercicio_t1.py
    │       02_ejercicio_t1.py
    │
    └───ejercicios_no_resueltos     # Ejercicios no resueltos
            01_ejercicio_t1.py
            02_ejercicio_t1.py
```

## ¿Para qué es cada archivo y carpeta?
- **Tema - 'tema'**: Es el nombre de la carpeta.
- **README - 'README.md'**: En este archivo está toda la explicacion mas "detalladada" sobre el tema visto y los ejercicios. Todos los temas tienen un 'README.md' independiente.
- **Apuntes - 'tema/apuntes'**: Apuntes y ejemplos con comentarios en los archivos dentro de /apuntes (cada tema tiene una carpeta 'apuntes').
- **Ejercicios - '/ejercicios'**: En esta carpeta encontraras los ejercicios resueltos relacionados al tema visto.
- **Ejercicios no resueltos - 'tema/ejercicios_no_resueltos'**: En esta carpeta encontraras los ejercicios aún no resueltos relacionados al tema visto.

## Instrucciones
Para empezar a trabajar con el proyecto en tu maquina de forma local, simplemente clona el repositorio y empieza a explorar los archivos dentro de cada carpeta de temas. Puedes resolver los ejercicios propuestos.
```bash
git clone https://github.com/KalebCxDev/python-desde-cero.git
```
